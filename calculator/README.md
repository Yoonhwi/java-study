## Java 계산기 프로젝트

### 📋 프로젝트 소개

이 프로젝트는 **Java**와 **Swing GUI**를 활용하여 간단한 계산기를 구현한 프로젝트입니다. Java의 기본적인 문법을 학습하는 데 목적이 있습니다.

---

### 📂 주요 기능

1. **사칙연산**: 덧셈, 뺄셈, 곱셈, 나눗셈 기능 제공
2. **클리어 버튼**: 입력값과 결과값 초기화
3. **실시간 계산**: 버튼 클릭 시 결과 출력
4. **GUI 구성**: 직관적인 UI로 사용 편의성 증대

---

### 🛠️ 기술 스택

- **언어**: Java
- **GUI**: Swing
